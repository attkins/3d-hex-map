# 3D hex map

![An experiment with hex maps and webgl](./docs/screenshot.png)


Just playing around with Three.js and TypeScript.

## Installation

This project uses TypeScript and Webpack. So installation is pretty simple:

```bash
  npm install
  npm start
```
